Untangling Practical Byzantine-Fault Tolerance

~~~~~~~~~~~~~~~


Recent advances in blockchain technology has brought forth interest in the design of Byzantine-Fault Tolerant (BFT) protocols. At the core of any blockchain application is a BFT consensus protocol. This protocol states the following: Given a client request and a set of replicas, some of which could be byzantine, the non-faulty replicas would agree on the order for this client request.
